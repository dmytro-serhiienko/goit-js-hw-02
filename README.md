# goit-js-hw-02

### Про модуль

## Цей модуль допомагає закріпити практичні навички роботи з умовами, логікою та циклами в JavaScript:

    •	Розуміння принципів роботи розгалужень: інструкції if, оператор switch, тернарний оператор.
    •	Використання логічних операторів: &&, ||, !.
    •	Застосування методів рядків: slice(), toLowerCase(), toUpperCase(), includes(), startsWith(), endsWith(), indexOf(), trim().
    •	Робота з циклами: while, do…while, for.
